# pyql

## Usage
    pyql.py [-h] [-d DATABASE] [-n NEW]

## Optional Arguments
    -h, --help                          show this help message and exit
    -d DATABASE, --database DATABASE    database to query from
    -n DATABASE, --new DATABASE         create a new database
